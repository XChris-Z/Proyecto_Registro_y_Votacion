## Development

When starting the dev server, use background mode:

```
astro dev --background
```

Manage the background server with `astro dev stop`, `astro dev status`, and `astro dev logs`.

## Despliegue y Control de Versiones (Automático)

- **Commit y Push Automático:** Siempre que termines de arreglar, modificar o mejorar algo en el código (una vez verificado que compila o funciona correctamente), debes realizar un `git commit` descriptivo y hacer un `git push origin main` automáticamente sin esperar a que el usuario lo solicite de nuevo, para que **Vercel** se actualice al instante con los últimos cambios.
